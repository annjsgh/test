# 使用说明
1. 克隆项目到本地
2. 安装Miniconda（推荐）或conda
```bash
https://docs.conda.io/en/latest/miniconda.html
```
3. 创建环境并安装依赖库
```
conda env create -f conda-environment.yml
```
4. 激活虚拟环境
```
conda activate software
```
5. 启动后端服务
```
cd backend
python app.py
```
6. 启动前端服务
```
start d:\Mycode\Software_Engineering_Code\frontend\index.html
```
